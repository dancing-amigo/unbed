# STEP4: アラーム設定UIを作る

## 目的
- ユーザーが MVP に必要なアラーム設定を完了できる画面を実装する。

## 実装内容
- Jetpack Compose で設定画面を作る。
- 時刻ピッカーを実装する。
- 曜日選択 UI を実装する。
- アラーム有効/無効トグルを実装する。
- 保存操作時に repository 更新と次回予約処理を呼ぶ。
- 現在設定されているアラーム内容を画面へ反映する。

## 完了条件
- アプリ起動後に時刻と曜日を設定して保存できる。
- 保存結果が永続化され、再起動後も反映される。
- 有効化すると次回アラーム予約まで繋がる。

## 注意点
- MVP は 1 件だが、一覧 UI に寄せすぎず、将来複数件化しやすいコンポーネント構造にする。
